################################################
1. Environment used:
- Python 3.8
- pandas   (install command: pip -install pandas)
- numpy    (install command: pip -install numpy)


################################################
2. How to run the program ?
- Input linear programming problem in input.txt
- Run main.py
- Check solution.txt and log.txt for the final answer and intermediate proccesses


################## important ###################
3. How to input question in input.txt

Example of input format
---------------------------------------------
type:
"maximisation"  

objective_function:
"x_1 + 2x_2" 

constraints:
"x_1 <= 4"
"x_2 <= 3"
----------------------------------------------
Note:
- All input contents should be placed within quotation marks
- ALL variables should be named as "x_" series, i.e. (x_1, x_2, x_3, ...)
- Add only one constraint per line


################################################
4. File Interpretation

- solution.txt : 
The final answer of the problem as well as variable type analysis will be stored there

- log.txt :
Each pivoting process will be stored there

For more details please refer to the original files
